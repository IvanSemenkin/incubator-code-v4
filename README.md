# Инкубатор 4.0 (Qubo)

## О проекте
**Инкубатор 4.0** – это проект модернизации существующих инкубаторов до уровня современных IoT-устройств. Он позволяет превратить обычный инкубатор в "умный" гаджет с возможностью контроля через веб-интерфейс и Telegram.

Проект открыт для всех, кто хочет повторить или доработать его под свои нужды.

## Возможности
- Автоматическая терморегуляция
- Контроль влажности
- Автоматический переворот яиц
- Переключение режимов работы
- Веб-интерфейс для удаленного управления
- Поддержка Telegram-уведомлений (в планах)
- Интеграция с голосовым помощником Алиса (в планах)

## Используемые технологии
### Аппаратная часть
- **ESP32** – микроконтроллер
- **DHT22** – датчик температуры и влажности
- **SSD1306** – OLED-дисплей
- **Электронные ключи и реле** – управление нагревателем, вентилятором, мотором

### Программная часть
- **MicroDot** – серверная часть веб-интерфейса
- **Python** – язык программирования
- **Sprint Layout** – проектирование печатных плат
- **AutoCAD 2025** – разработка корпуса

## Установка и запуск
### Требования
- ESP32 с установленным MicroPython
- Настроенная среда для загрузки кода (например, Thonny или esptool)
- Интернет-соединение для работы веб-интерфейса

### Инструкция по запуску
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/IvanSemenkin/incubator-code-v4.git
   cd incubator-code-v4
   ```
2. Установите зависимости (если требуются).
3. Загрузите код на ESP32.
4. Подключитесь к веб-интерфейсу для управления.
